Install sphinx-bootstrap-theme: ` cd ~/Development/beaglesystems git clone git@github.com:dayjaby/sphinx-bootstrap-theme cd sphinx-bootstrap-theme sudo python3 setup.py install `

Modify the CSS: ` cd ~/Development/beaglesystems git clone git@github.com:dayjaby/bootstrap cd bootstrap npm install npm install grunt ... modify files in ./less/ ... ./node_modules/.bin/grunt dist cp -r dist/* ~/Development/beaglesystems/sphinx-bootstrap-theme/bootstrap/static/bootstrap-3.4.1/ `

Create the documentation: ` sphinx-apidoc -f -o source/mqtt_bridge ../src/mqtt_bridge/src/mqtt_bridge make clean && make html `
